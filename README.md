![CD Status](https://github.com/Lucas127128/better-event/workflows/npm-publish/badge.svg)

A better event emitter with type safety! (Inspired by [emittery](https://github.com/sindresorhus/emittery))

## Development

- Install dependencies:

```bash
bun install
```

- Run the unit tests:

```bash
bun run test
```

- Build the library:

```bash
bun run build
```
